demo.m: demo script with 3D simulation setting

need to include toolboxes: 'tensor_toolbox_2.6' and 'TensorReg'

ten_qreg.m: tensor quantile regression
ten_regqreg.m: regularized tensor quantile regression
admm_rqr.m: admm solver for penalized quantile regression

generate_signal.m: data generation for simulation
cauchy1000_100_1.mat: generated Cauchy errors

rq_fnm.m: interior point algorithm for quantile regression download from http://www.econ.uiuc.edu/~roger/research/rq/rq.html 
lp_fnm.m: internal function for rq_fnm.m
bound.m: internal function for rq_fnm.m
